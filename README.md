# Odin-Recipes: A Review of Basic HTML

This project is an application of the basic HTML skills that I have learned in The Odin Project Foundations Course.  
The project includes:
- A properly formatted project directory
- A basic site using only HTML
- At least 3 images
- At least 2 pages linked to the index
